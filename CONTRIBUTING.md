# Contributing Guidelines

Thank you for taking the time to contribute to this project! 

All contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome.

---

## 🐛 How to Report a Bug

1. **Search existing issues** to make sure the bug hasn't already been reported.
2. If it's a new issue, **open a new bug report** on GitHub.
3. Clearly describe the problem, including steps to reproduce the bug, expected behavior, and actual behavior.
4. Include screenshots or error logs if applicable.

## 💡 How to Suggest an Enhancement or Idea

1. Check the existing project issues to see if the feature has already been suggested.
2. Open a new issue and choose the **Feature Request** template (or tag it as an enhancement).
3. Explain the user story: *Why* is this feature needed, and *how* should it work?

## 🛠️ How to Submit Changes (Pull Requests)

To contribute code, documentation fixes, or enhancements directly, please follow the standard Git workflow:
