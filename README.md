# Simple Interest Calculator

A lightweight, command-line application that calculates simple interest based on principal amount, annual interest rate, and time period. This project was built as part of a final Git and GitHub hands-on assignment to demonstrate version control workflows.

---

## 🚀 Features

* **Quick Calculations:** Instantly computes simple interest and the total final amount.
* **User-Friendly CLI:** Prompts the user for straightforward inputs.
* **Educational Purpose:** Developed to practice forking, cloning, staging, committing, and creating pull requests in Git/GitHub.

---

## 🧮 Formula Used

The application calculates simple interest using the standard formula:

$$A = P(1 + rt)$$

Where:
* $A$ = Total Final Amount (Principal + Interest)
* $P$ = Principal Amount
* $r$ = Annual Interest Rate (as a decimal)
* $t$ = Time Period (in years)
